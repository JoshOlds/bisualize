Bisualize is a corporate tool to vizualize employee progress, skills, and experience

Bisualize is a capstone project for CodeWorks Boise and was designed and coded by Joshua Olds, Meredith Ganz, Dustin Patterson, and Mark Ohnsman.

The relationships of the data in Bisualize:

The organization Bisualize's backend relationships is tied together by positions. Positions represent a physical spot on the organization tree, each position must have a manager position, and it can have reports underneath it.

A position has an employee and a job property that can be changed. This allows for employees to be moved from one team to another, or be promoted. It also means that the job title associated with the position can change.

Employees have a job and have badges which are lists that hold a title and description. These are used as references to positions and employees but are not reliant upon them.

Badges are used to both reward employees and track their experience within the company. For example, if an employee is cross-trained he/she may earn a badge indicating that they have completed that training.

Badges can be a useful tool during employee reviews or the promotion process as indicators of employees' accomplishments.

    • Description

    “My Page” is a more detailed view of a specific employee within the company. You can see the employee’s name, title, badges earned, their manager, and employees they manage.

    Instructions

    Badges
    • If you hover over the badge icons, the name and description of the badge will appear.
    Navigate Employees
    • To navigate to another employee, you can click on their picture and it will load their “My Page” view.

    Description

    “Org Chart” provides a 30,000-foot view of the employees within the company. You can quickly drill-down into departments and see all the employees that are part of a department.

    Instructions

    Drill-down functionality
    • When “Org Chart” first loads, it defaults to the company’s executive level employees. If the employee’s card has a blue glow, it means that they manage a department. By clicking on the text portion of their card, the “Org Chart” will expand and list that employee’s direct reports.
    Link to “My Page”
    • If you would like to view more detailed information for a specific employee, you can click on their picture and it will load that employee’s “My Page” view.

    Description

    The “Admin” view allows administrators within an organization to make changes to the company’s structure. Through the “Admin” view, you can create, edit or delete employees, positions, jobs, and badges.

    Instructions

    Hire a new employee
    • Step 1 - Navigate to the “Admin” page, select the “Employees” tab and select the “New” sub-tab. Type the employee’s name. Paste in a web link for their picture in the appropriate field. The new employee will be added to the database once you click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the employee was added or if there was an error. If the employee is filling an existing position, skip to Step 3.
      NOTE : An employee’s name is required to create an employee. If you do not add a picture URL, a default image will be automatically added for the employee.

    • Step 2 (SKIP THIS STEP IF THE EMPLOYEE IS FILLING AN OPEN POSITION) – Click over to the “Position” tab and make sure that the “New” sub-tab is still selected. Using the drop-down, select the “Manager’s Position” to select who the manager will be for the new position. Using the drop-down, select a Job title for the new position. Click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
      NOTE : If the job does not exist, you will need to create a new job through the “Jobs” tab. The instructions can be found in the “Create a new job” section of the instructions.

    • Step 3 – On the “Employees” tab, select the “Edit” sub-tab. From here, select the new employee from the “Employee Name” dropdown. Navigate down to the “Available Position” section and select an open position. Click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
      NOTE : Only available positions will show in the drop-down. If nothing is displaying in the drop-down, go back to step 2 and create a new position.
    Create a new job
    • Step 1 – Navigate to the “Admin” page, select the “Jobs” tab and select the “New” sub-tab. Type in the new job’s official title in the “Job Title” field. Add the job’s responsibilities and requirements to the “Job Description” field. Once you are satisfied with the level of detail needed for the job, click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
    Create a new badge
    • Step 1 – Navigate to the “Admin” page, select the “Badges” tab and select the “New” sub-tab. Type in the new badge title in the “Badge Title” field. Paste in a web link for the badge’s picture in the appropriate field. Add the badge’s description to the “Badge Description” field. Once you are satisfied with the level of detail around the badge, click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
    Edit an employee’s name
    • Step 1 - Navigate to the “Admin” page, select the “Employee” tab and select the “Edit” sub-tab. Find the employee you need to edit using the “Employee Name” dropdown menu. New options will populate, but you only need to modify the text inside the “Edit Name” field on the form. Make the required changes. Once you have made the required change, click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
    Promote an employee
    • Step 1 - Navigate to the “Admin” page, select the “Employee” tab and select the “Edit” sub-tab. Find the employee you need to promote using the “Employee Name” dropdown menu and new options will populate. Using the “Available Position” dropdown, select the new position that the employee is being promoted to. Click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
      NOTE : This will change will automatically update the employee’s job to the job listed on the position.
    Award an employee with a badge
    • Step 1 - Navigate to the “Admin” page, select the “Employee” tab and select the “Edit” sub-tab. Find the employee you need to award with a badge using the “Employee Name” dropdown menu. New options will populate, but you should only use the “Add Badge” dropdown at the bottom of the form. Using the dropdown, select the badge the employee has earned. Click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
    Edit a badge
    • Step 1 - Navigate to the “Admin” page, select the “Badges” tab and select the “Edit” sub-tab. Find the badge you need to edit using the “Badge Title” dropdown menu. You will now have options to edit the badge’s name, picture, and description. Make the required changes. Once you are satisfied with the changes, click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
    Terminate an employee
    This action cannot be undone
    • Step 1 - Navigate to the “Admin” page, select the “Employee” tab and select the “Edit” sub-tab. Find the employee you need to terminate using the “Employee Name” dropdown menu. New options will populate, but you only need to select the “Terminate Employee” box at the bottom of the form. Click the “Submit” button. A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
      NOTE : This action CANNOT be undone.

A message at the bottom of the screen will display momentarily notifying you if the action was successful or if there was an error.
